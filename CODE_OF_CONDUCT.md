# Code of Conduct

Pulse follows the GNOME Code of Conduct for all project spaces and interactions.

Official policy:
- https://conduct.gnome.org/

Expected behavior includes:
- being respectful and patient
- assuming good intent while communicating clearly
- giving constructive technical feedback without personal attacks
- welcoming contributors with different backgrounds and experience levels

Unacceptable behavior includes:
- harassment, intimidation, or discrimination
- insults, personal attacks, or hostile language
- publishing private information without permission
- disruptive behavior that makes collaboration unsafe or exhausting

## Reporting

If you experience or witness inappropriate behavior related to this project, report it privately to the project maintainer through a private contact channel on the repository host.

If an incident happens in GNOME-managed spaces, follow the official GNOME reporting process:
- conduct@gnome.org
- https://conduct.gnome.org/

All reports should be handled as discreetly as possible.
